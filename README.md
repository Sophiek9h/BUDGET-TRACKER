BUDGET TRACKER

This is an application that helps you keep track of your expenses, ie to calculate money being spent, how much it cost, and what remains afterwards.
This webpage was built in HTML, CSS (Bootstrap included), and JS only. It's a single webpage and is responsive across all devices.


HOW TO SET IT UP

- clone this repository if you're on github or download it
- open on VS code and run on a browser 

HOW TO USE

1) load the page
2) input the total amount you want to spend
3) input the name of the item you want to get
4) enter the price 
5) enter the amount of item you want to get
6) add the item using the "add product" button
7) you can use the edit button and/or the delete button to remove items from your list of expenses
